Approach

The project was very developer focused. It involved me learning how the blockchain has become programmable with smart contracts, little pieces of automatically-executing code running on the blockchain. It also involved creating a hosted website where people could visit and learn about the project.

Goals

The main goal of the project was to learn more about cryptocurrency, the blockchain, NFT's and improve my skills as developer. As this was one of my biggest solo projects, one of the biggest challenges was being able to develop a solution that had the functionality the project needed.

To ensure the outcome was successful, the website and designs were tested with defined users over several iterations.

Outcome

The outcome was a fully responsive website and a random algorithmatically generated composite PNG image maker built in Node JS.
